# 🚗 Uber Ride Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing Uber ride data to understand booking patterns, revenue trends, cancellation behavior, and customer satisfaction. The analysis is performed using Python for data processing and Power BI for interactive dashboard visualization.

---

## 🎯 Objective

To perform end-to-end data analysis on Uber ride data and develop an interactive dashboard that provides actionable insights for improving operational efficiency and customer experience.

---

## 📊 Dataset Description

The dataset contains detailed ride-sharing information including:

* Booking details (ID, date, time)
* Customer and vehicle information
* Pickup and drop locations
* Ride distance and booking value
* Booking status (completed, cancelled, etc.)
* Cancellation reasons (customer/driver)
* Payment methods
* Customer and driver ratings

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Power BI
* Jupyter Notebook

---

## 🔄 Project Workflow

1. Data Collection
2. Data Cleaning

   * Handled missing values using logical and statistical methods
   * Removed duplicates
   * Converted data types
3. Feature Engineering

   * Extracted Hour, Day, and Month from datetime
4. Exploratory Data Analysis (EDA)

   * Demand analysis
   * Revenue analysis
   * Cancellation analysis
   * Customer satisfaction analysis
5. Dashboard Development

   * Built interactive Power BI dashboard with KPIs and filters

---

## 📈 Key Insights

* Peak ride demand observed during evening hours
* Significant cancellation rate identified, especially in certain vehicle types
* Digital payment methods are widely preferred
* Revenue varies across vehicle categories
* Customer ratings indicate varying satisfaction levels across services

---

## 📊 Dashboard Features

* KPI Cards (Total Bookings, Revenue, Cancellation Rate, Ratings)
* Interactive filters (Date, Vehicle Type, Payment Method)
* Booking status breakdown
* Ride trends over time
* Revenue distribution
* Customer insights
